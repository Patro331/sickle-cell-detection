# Deep Learning for Automated Sickle Cell Detection in Peripheral Blood Smears

An explainable deep learning project for automated sickle cell disease 
detection using African clinical microscopy data, built as part of 
MSB7216: Deep Learning for Health Data.

**Author:** Okidi Patrovas Gaabriel | 2025/HD07/26020U

## Project Overview
This project applies transfer learning (EfficientNet-B0 and ResNet-50) 
to detect sickle cells in peripheral blood smear images, with Grad-CAM 
explainability and a Gradio deployment demo targeting low-resource 
clinical settings in Uganda.

## Datasets
- Mendeley Peripheral Blood Cell Images — Acevedo et al. (2020)
- Tushabe Uganda Sickle Cell Microscopy Dataset (2024)

## Project Structure
- `data/` — Raw and processed image datasets
- `notebooks/` — Jupyter notebooks for EDA, training, and evaluation
- `src/` — Python source code (dataset loader, model, training, evaluation)
- `models/` — Saved model weights
- `reports/` — Final project report
- `figures/` — EDA plots and Grad-CAM visualisations
- `app/` — Gradio deployment application
